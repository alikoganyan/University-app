# Simple app with Laravel 5.4, AngularJS and Twitter Bootstrap. #

### Run following commands to configure and start project ###
* Install dependencies *composer install*
* Create .env file *cp .env.example .env*
* Generate key *php artisan key:generate*
* config homestead * php vendor/bin/homestead make*
* To run project make *vagrant up*
* Connect to project’s  VM *vagrant ssh*
* Switch to your project directory (usually it’s *Code/project-name*)
